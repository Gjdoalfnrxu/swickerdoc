What is tsq?

A semi-naive Datalog query engine over tree-sitter TypeScript/JavaScript ASTs.

  • Extracts facts (relations) from source files via the TypeAwareWalker
  • Evaluates QL queries (CodeQL-compatible syntax) against those facts
  • Ships a taint-tracking pipeline — 16 local flow steps + interprocedural
  • Bridge layer provides CodeQL-compatible imports (DataFlow::Node, TaintTracking::Configuration)

Key constraint: v1 CallResultSym intentionally empty — no demand-driven call graph.

Workaround: library summaries (InterFlow rules) cover 25+ frameworks explicitly.
